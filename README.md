# LYC homes Web

## 1. 安装依赖（ Yarn ）


- 使用以下命令来安装项目的依赖：
```bash
yarn install
```

## 2. 启动项目

> 你可以使用 Yarn 或 npm 来启动项目：

```bash
yarn start
```

## 3.查看本地是否下载 node.js 
> 可以在 コマンドプロンプト 里查看node.js版本 如果无法显示说明电脑里没下载 需要重新下载安装node.js


## 4.npm/yarn 启动不了的时候 
> 把node_modules 删除 重新执行 yarn install 或者 npm install
